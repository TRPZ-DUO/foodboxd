# 🍽️ Foodboxd

> Descubra, avalie e organize suas experiências gastronômicas.

O **Foodboxd** é uma plataforma inspirada no Letterboxd, mas voltada para o universo da gastronomia. O objetivo é permitir que usuários registrem os pratos e restaurantes que experimentaram, criem listas, publiquem avaliações e descubram novas experiências através da comunidade.

---

## ✨ Funcionalidades

### Usuários

* Cadastro e autenticação
* Perfil personalizado
* Foto de perfil
* Estatísticas de avaliações
* Histórico gastronômico

### Avaliações

* Avaliar pratos
* Avaliar restaurantes
* Nota de 0.5 a 5 estrelas
* Comentários e opiniões
* Fotos da experiência

### Organização

* Diário gastronômico
* Favoritos
* Lista de desejos (Wishlist)
* Listas personalizadas
* Restaurantes visitados

### Social

* Seguir usuários
* Curtidas em avaliações
* Comentários
* Feed de atividades
* Compartilhamento de listas

### Descoberta

* Buscar restaurantes
* Buscar pratos
* Categorias culinárias
* Rankings
* Recomendações
